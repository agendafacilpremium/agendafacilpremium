PASSO A PASSO PARA LEIGO – COMO SUBIR NO RENDER

1. Acesse https://render.com e crie uma conta (ou entre com sua conta).
2. Clique em "New Web Service".
3. Escolha a opção: Deploy from GitHub OU "Upload manualmente".
4. Se for subir manualmente:
   - Extraia este .zip
   - Entre na pasta extraída e compacte os arquivos novamente em .zip (não inclua a pasta, só os arquivos dentro dela)
   - Faça o upload no Render
5. Quando pedir o comando de start, coloque:
   yarn start
   ou
   npm start
6. Pronto! O site será carregado com o backend funcionando.
7. Aguarde até 50 segundos se estiver na versão gratuita.
